添加规则：

1. 段与段之间用 "-----" 五个连线符区分开(不包含双引号)，其中一张图片，一个视频都算单独的一段。

2. 添加图片的规则： "imgs/" + "图片的名字"，如 "imgs/电梯.png" (不包含引号，以下所有格式相关都不包含引号)。

    (注意： 当相邻段落都为图片时，这些图片会显示在同一行。)

3. 图片格式限制： *.png, *.jpg, *.gif, *.jpeg

4. 添加视频的规则： "videos/" + "视频的名字"，如 "videos/手表.mp4"。

5. 视频格式限制： *.mp4, *.ogg

6. 添加图片或视频的解释短语规则： "define/" + "解释短语"，如 "define/燃气灶的工作原理"。
